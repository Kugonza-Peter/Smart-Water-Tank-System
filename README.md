

## Project Overview
This is an IoT-based Smart Water Tank System developed using ESP32 and an ultrasonic sensor.

The system automatically monitors water levels and controls a water pump (simulated using LED).

## Components Used
- ESP32
- HC-SR04 Ultrasonic Sensor
- Relay Module
- LED (Pump simulation)
- Jumper wires

## Features
- Automatic water level monitoring
- Automatic pump ON/OFF control
- Serial monitor output
- IoT-ready architecture
